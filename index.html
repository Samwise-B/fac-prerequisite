Why I'm Applying to Founders & Coders

I studied computer science in school at GCSE and A-Level. During that time, I would mostly code in Python. I also studied HTML, CSS and JavaScript a bit, but I didn’t build a website. The curriculum wasn’t just programming. I studied computer theory, which included learning the basics of how a computer works (CPU, RAM, GPU, HDD, SSD), as well as learning about software and networking. Throughout A levels I became more interested in computer science. When A levels finished I decided not to go to university straight away. Instead, I started looking at online programming courses.

In February this year I started CS50x which helped me to connect the dots between the theory I had learnt and my programming knowledge by coding in C. By the end of the course I had built my own web app using the Python framework Flask. I used bootstrap for the responsive design, which was my first time making my own web app. You can find my final project here.

I finished CS50 at the end of June and it left me enthusiastic to work on bigger projects. I came back from a holiday at the end of July and I looked into FAC at the beginning of August.

After checking out the website, as well as coming to a meetup, I got the impression that FAC would be a great way to further my understanding of how more complicated web applications work in an environment where I can get hands on experience developing projects in a group. Doing online courses was a positive experience, but getting input from peers would help my work rate and group work seems to be a crucial ability that employers look for. FAC seems like a community of eager developers coming together to learn how to solve real problems. The tech for better programme also seems like a smart way to try and make a positive impact on the world.

What I Gained From the Prerequisites

I started the prerequisites on the 8th of August. Despite starting late, I still wanted to try and apply for the winter cohort. The prerequisites were a good opportunity for me to revisit vanilla CSS and JavaScript, which definitely helped to consolidate my knowledge of web programming languages. CodeWars was also useful because I got to practice areas of JavaScript I wasn’t so familiar with like some of the common data type modules (array.map, array.filter, etc), as well as RegEx.
